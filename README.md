# Mood-Up Backend

Backend desarrollado para el proyecto Mood-Up, encargado de la gestión de los usuario, manejo emocional y recomendación de contenido multimedia y retos personalizados según el estado emocional del usuario.

Este proyecto fue desarrollado como parte de un proyecto académico universitario.

## 🚀 Tecnologías utilizadas

- C#
- ASP.NET
- MySQL
- MySQL Connector
- phpMyAdmin
- JWT Authentication

## ⚙️ Funcionalidades principales (APIs)

- Registro e inicio de sesión
- Autenticación mediante JWT
- Registro e historial de emociones
- Recomendación de contenido multimedia según la emoción del usuario
- Recomendación de retos y actividades según la emoción del usuario

## 🗃️ Base de datos

El backend de Mood-Up utiliza MySQL como sistema gestor de dase de datos y phpMyAdmin como herramienta de administración

## 🗂️ Estructura del proyecto

````
  MoodUP_final/
    ├── Controlers/
    ├── DBO/
    ├── Data/
    ├── Migrations/
    ├── Models/
    ├── Properties/
    └── Services/
````

## 🔗 Endpoints principales

- Registro de usuarios
- Inicio de sesión
- Registro de emociones
- Historial de emociones
- Filtrar películas y series por emoción
- Filtrar retos por emoción

## 🛡️ Seguridad

- Autenticación con JWT
- Escriptación de contraseñas
- Protección de rutas
